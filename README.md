# NeatPDF

Aplicación web fullstack para procesar archivos PDF — unir, separar, comprimir y convertir a imagen.

- **Frontend** — React 19 + Vite desplegado en Vercel → [Ver demo](https://neat-pdf-eta.vercel.app)
- **Backend** — FastAPI + PostgreSQL desplegado en Render

## Correr con Docker

```bash
docker compose up --build
```

- Frontend: http://localhost  
- Backend API: http://localhost:8000/docs
